# Деплой проекту Абетка (ABC Helper) на Ubuntu за допомогою Portainer

Цей файл містить покрокову інструкцію, як розгорнути цей проект на вашому сервері Ubuntu, використовуючи Docker та Portainer.

## Передумови

1.  **Сервер Ubuntu** з встановленим Docker та Docker Compose.
2.  **Portainer**, встановлений та запущений.
3.  **API Key** від Google Gemini (для озвучення та перевірки малюнків).

---

## Варіант 1: Деплой через GitHub (Stacks) - Рекомендовано

Це найкращий спосіб, оскільки Portainer автоматично завантажить код з репозиторію.

1.  Зайдіть у **Portainer**.
2.  Перейдіть у **Stacks** -> **Add stack**.
3.  Оберіть **Repository**.
4.  Заповніть поля:
    *   **Name:** `abc-helper` (або інше на ваш вибір).
    *   **Repository URL:** `https://github.com/Nicks-jpg/Alphabet-teacher` (або посилання на ваш форк).
    *   **Repository reference:** `refs/heads/main` (або назва вашої гілки).
    *   **Compose path:** `docker-compose.yml`.
5.  У розділі **Environment variables** додайте:
    *   `API_KEY`: Ваш ключ від Google Gemini.
6.  Натисніть **Deploy the stack**.

Portainer завантажить код, збудує Docker-образ (це займе кілька хвилин) і запустить контейнер.

---

## Варіант 2: Локальний деплой (якщо код вже на сервері)

Якщо ви вже склонували репозиторій на сервер:

1.  Створіть файл `.env` у папці проекту:
    ```bash
    echo "API_KEY=ваш_ключ_тут" > .env
    ```
2.  Запустіть Docker Compose:
    ```bash
    docker-compose up -d --build
    ```

## Перевірка роботи

1.  Відкрийте у браузері: `http://IP-адреса-вашого-сервера:3030`
2.  Щоб перевірити звук та налаштування, перейдіть на сторінку:
    `http://IP-адреса-вашого-сервера:3030/test-sound.html`

## Виправлення проблем

### Немає звуку
1.  Переконайтеся, що ви додали правильний `API_KEY`.
2.  Відкрийте `test-sound.html` і натисніть "Run Test". Подивіться на лог помилок.
3.  Якщо Gemini не працює, система автоматично переключиться на браузерний синтез мови (Web Speech API).

### Оновлення версії
Якщо ви внесли зміни в код на GitHub:
1.  Зайдіть у **Portainer** -> **Stacks** -> **abc-helper**.
2.  Натисніть **Editor**.
3.  Натисніть **Update the stack** і обов'язково поставте галочку **Re-pull image and redeploy**.
